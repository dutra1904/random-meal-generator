Sobre o Random Meal Generator

Este aplicativo exibe refeições aleatórias da API TheMealDB. Ao clicar no botão, uma nova receita é buscada com nome, imagem, ingredientes e instruções de preparo. Quando disponível, um vídeo do YouTube é incorporado para auxiliar no preparo.

Clique no botão acima para gerar uma receita
